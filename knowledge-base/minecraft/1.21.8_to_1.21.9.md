## Version: 1.21.8 → 1.21.9

> **Java requirement:** Java 21 (unchanged)
> **Source:** https://minecraft.wiki/w/Java_Edition_1.21.9

---

### Class Moves

**PlayerSkin moved package:**
- old: `net.minecraft.client.resources.PlayerSkin`
- new: `net.minecraft.world.entity.player.PlayerSkin`
- source: Verified via compile (MC 1.21.9)

**PlayerSkin API change:**
- old: `playerSkin.texture()` → returns `ResourceLocation`
- new: `playerSkin.body().texturePath()` → returns `ResourceLocation`
- source: Verified via compile (MC 1.21.9)

---

### Input API Changes

**Screen.keyPressed:**
- description: `keyPressed` signature changed to use `KeyEvent` wrapper class
- before: `public boolean keyPressed(int keyCode, int scanCode, int modifiers)`
- after: `public boolean keyPressed(KeyEvent event)` — use `event.key()` for key code
- source: Verified via compile (MC 1.21.9)

**Required import:**
- Add `import net.minecraft.client.input.KeyEvent;`

---

### KeyMapping Category (NeoForge)

- description: `KeyMapping` constructor no longer accepts `String` category in NeoForge 1.21.9+; requires `KeyMapping.Category`
- before: `new KeyMapping("key.mymod.action", type, key, "key.categories.misc")`
- after: `new KeyMapping("key.mymod.action", type, key, KeyMapping.Category.register(ResourceLocation.fromNamespaceAndPath("mymod", "mymod")))`
- note: This change is in NeoForge 1.21.9; Fabric moved to `KeyMapping.Category` in 1.21.10
- source: Verified via compile (NeoForge 21.9.x)

---

### NeoForge FMLEnvironment

- description: `FMLEnvironment.getDist()` method replaced by `FMLEnvironment.dist` field
- before: `FMLEnvironment.getDist()`
- after: `FMLEnvironment.dist`
- source: Verified via compile (NeoForge 21.9.x)

---

### Common Migration Fixes

1. Update `PlayerSkin` import: `client.resources` → `world.entity.player`
2. Update `playerSkin.texture()` → `playerSkin.body().texturePath()`
3. Replace `keyPressed(int keyCode, int scanCode, int modifiers)` → `keyPressed(KeyEvent event)`
4. Add `import net.minecraft.client.input.KeyEvent;`
5. Replace `keyCode` references in body → `event.key()`
6. Update `super.keyPressed(keyCode, scanCode, modifiers)` → `super.keyPressed(event)`
7. (NeoForge) Replace `FMLEnvironment.getDist()` → `FMLEnvironment.dist`
8. (NeoForge) Update `String` KeyMapping category → `KeyMapping.Category.register(...)`
