## Version: 1.20.6 → 1.21

> **Java requirement:** Java 21 (unchanged)
> **Source:** https://minecraft.wiki/w/Java_Edition_1.21

---

### New Classes

- `net.minecraft.client.DeltaTracker` — replaces `float tickDelta` in HUD render callbacks
- source: Fabric API porting guide; Mojang mappings

---

### HUD Render Callback (Fabric)

- description: `HudRenderCallback` callback signature changed — `float tickDelta` replaced by `DeltaTracker`
- before: `HudRenderCallback.EVENT.register((poseStack, tickDelta) -> ...)`
- after: `HudRenderCallback.EVENT.register((graphics, deltaTracker) -> ...)`
- note: To get the equivalent of `tickDelta`, use `deltaTracker.getGameTimeDeltaPartialTick(true)`
- source: Fabric API changelog for 1.21

---

### Identifier Constructor Change

- description: `Identifier` constructor made protected; must use factory method
- before: `new Identifier("namespace", "path")` or `new Identifier("namespace:path")`
- after: `Identifier.of("namespace", "path")` or `Identifier.of("namespace:path")`
- vanilla shorthand: `Identifier.ofVanilla("name")` for minecraft namespace
- safe parsing: `Identifier.tryParse("namespace", input)` for untrusted input
- source: https://fabricmc.net/2024/05/31/121.html

---

### Data Pack Path Singularization

All data pack directory names changed from plural to singular:

| Old (1.20.x) | New (1.21+) |
|---|---|
| `tags/blocks` | `tags/block` |
| `tags/entity_types` | `tags/entity_type` |
| `tags/functions` | `tags/function` |
| `advancements` | `advancement` |
| `functions` | `function` |
| `item_modifiers` | `item_modifier` |
| `loot_tables` | `loot_table` |
| `predicates` | `predicate` |
| `recipes` | `recipe` |
| `structures` | `structure` |

Note: The `tags` directory itself remains plural.
- source: https://fabricmc.net/2024/05/31/121.html

---

### FabricDimensions API Removed

- description: `FabricDimensions.teleport()` removed
- before: `FabricDimensions.teleport(entity, serverLevel, target)`
- after: `entity.teleportTo(target)` where target is a `TeleportTarget`
- source: https://fabricmc.net/2024/05/31/121.html

---

### Common Migration Fixes

1. Add `import net.minecraft.client.DeltaTracker;` if using HUD callbacks
2. Update HUD callback lambda: `(poseStack, tickDelta)` → `(graphics, deltaTracker)`
3. Replace `tickDelta` uses inside callback body with `deltaTracker.getGameTimeDeltaPartialTick(true)`
4. Replace `new Identifier(` → `Identifier.of(`
5. Update data pack directory names (plural → singular, see table above)
6. Replace `FabricDimensions.teleport(` → `entity.teleportTo(`
7. Update `fabric-api` to `0.100.8+1.21`, `fabric-loader` to `0.15.11+build.1`
