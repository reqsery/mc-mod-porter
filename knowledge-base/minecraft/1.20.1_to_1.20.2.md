## Version: 1.20.1 → 1.20.2

> **Java requirement:** Java 17 (unchanged)
> **Source:** https://minecraft.wiki/w/Java_Edition_1.20.2
> **Note:** This is the version where Forge forked into NeoForge.

---

### API Changes (Forge → NeoForge)

- description: Forge forked into NeoForge at 1.20.2; package renamed from `net.minecraftforge` to `net.neoforged`
- impact: All Forge imports, annotations, and mod ID strings referencing `minecraftforge` must become `neoforged`
- source: https://neoforged.net/news/theproject/

### Class Moves

- old: `net.minecraftforge.*`
- new: `net.neoforged.*`
- source: NeoForge fork; https://neoforged.net/news/theproject/

### API Changes (Minecraft)

- description: `PlayerSkin` record introduced in 1.20.2 to wrap skin data
- impact: Replace `player.getSkinTextureLocation()` → `player.getSkin().texture()`
- source: https://minecraft.wiki/w/Java_Edition_1.20.2; Mojang mappings

---

### Common Migration Fixes

1. Find/replace `import net.minecraftforge` → `import net.neoforged`
2. Find/replace all `minecraftforge` strings → `neoforged`
3. Update `neoforge_version` in `gradle.properties` (NeoForge uses `20.2.x` format)
4. Replace `player.getSkinTextureLocation()` → `player.getSkin().texture()`
