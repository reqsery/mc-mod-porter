## Version: 1.20.4 → 1.20.5

> **Java requirement:** Java 21 (upgraded from Java 17 in 1.20.5)
> **Source:** https://minecraft.wiki/w/Java_Edition_1.20.5

---

### Class Renames

- old: `net.minecraft.network.FriendlyByteBuf`
- new: `net.minecraft.network.RegistryFriendlyByteBuf`
- note: Fabric uses `PacketByteBuf` — see below
- source: Mojang mappings; https://minecraft.wiki/w/Java_Edition_1.20.5

- old: `net.minecraft.network.PacketByteBuf` (Fabric/intermediary name)
- new: `net.minecraft.network.RegistryFriendlyByteBuf`
- source: Mojang mappings

---

### Networking API

- description: `RegistryFriendlyByteBuf` constructor now requires `RegistryAccess`
- impact: `new FriendlyByteBuf(Unpooled.buffer())` → `new RegistryFriendlyByteBuf(Unpooled.buffer(), player.registryAccess())`
- source: Mojang mappings

---

### NeoForge HUD API (1.20.5+)

- description: NeoForge HUD overlay system changed from `RegisterGuiOverlaysEvent` to `RegisterGuiLayersEvent`
- impact: Replace `RegisterGuiOverlaysEvent` import → `RegisterGuiLayersEvent`
- before callback: `(gui, graphics, partialTick, width, height) -> render(graphics, partialTick)`
- after callback: `(graphics, tickDelta) -> render(graphics, tickDelta)`
- source: NeoForge 1.20.5 release

---

### NeoForge Tick Event (1.20.5+)

- description: `TickEvent.ClientTickEvent` replaced by `ClientTickEvent.Post`
- before: `private void onTick(TickEvent.ClientTickEvent event) { if (event.phase == Phase.END) { ... } }`
- after: `private void onClientTick(ClientTickEvent.Post event) { ... }`
- source: NeoForge 1.20.5 release

---

### Common Migration Fixes

1. Replace `PacketByteBuf` / `FriendlyByteBuf` → `RegistryFriendlyByteBuf` everywhere
2. Update import: `net.minecraft.network.PacketByteBuf` → `net.minecraft.network.RegistryFriendlyByteBuf`
3. Update Java toolchain: `languageVersion = JavaLanguageVersion.of(21)`
4. Update NeoForge `RegisterGuiOverlaysEvent` → `RegisterGuiLayersEvent` if using NeoForge
5. Update `TickEvent.ClientTickEvent` → `ClientTickEvent.Post` if using NeoForge
