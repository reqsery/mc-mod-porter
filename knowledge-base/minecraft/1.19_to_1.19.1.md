## Version: 1.19 → 1.19.1

> **Java requirement:** Java 17 (unchanged)
> **Type:** Minor update — chat signing introduced
> **Source:** https://minecraft.wiki/w/Java_Edition_1.19.1

---

### API Changes

- description: Chat signing system introduced. Message event callback signatures changed.
- impact: `ClientReceiveMessageEvents.ALLOW_CHAT` callback signature changed
- source: https://fabricmc.net/wiki/tutorial:1.19_migration_guide

---

### Migration Notes

- If your mod handles chat messages or chat events, review your event handlers
- Update `fabric-api` to `0.58.5+1.19.1`
